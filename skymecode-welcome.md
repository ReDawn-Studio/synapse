**下一步操作**：

1. **克隆仓库**
```bash
git clone https://github.com/ReDawn-Studio/synapse.git
cd synapse
npm install
```

2. **查看任务详情**
打开 `docs/TASKS.md` 查看你认领的任务

3. **创建分支**
```bash
git checkout -b feature/frontend-ui
```

4. **开始开发**
- 参考 `docs/API.md` 了解 API 设计
- 使用 Vite + React + Tailwind 快速搭建

5. **提交代码**
```bash
git add .
git commit -m "feat: 创建前端项目和基础页面"
git push -u origin feature/frontend-ui
```

6. **创建 PR**
在 GitHub 上创建 Pull Request
