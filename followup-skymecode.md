**如何开始**：

1. **访问仓库**: https://github.com/ReDawn-Studio/synapse

2. **查看你的任务**: 打开 `docs/TASKS.md`
   - 阶段 5: 前端 UI（9 个任务）
   - 包括：React 项目搭建、登录页、频道列表、消息流等

3. **克隆仓库**:
```bash
git clone https://github.com/ReDawn-Studio/synapse.git
cd synapse
npm install
```

4. **创建你的分支**:
```bash
git checkout -b feature/frontend-ui
```

5. **开始开发前端**:
   - 使用 Vite + React + Tailwind
   - 参考 `docs/API.md` 了解 API 接口

6. **完成后提交 PR**:
```bash
git add .
git commit -m "feat: 创建前端 UI"
git push -u origin feature/frontend-ui
# 然后在 GitHub 上创建 Pull Request
```
